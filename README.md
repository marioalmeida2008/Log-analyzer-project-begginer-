# Log-analyzer-project-begginer-
Este é um projecto bastante simples para estudar análise de logs de Segurança.

Objectivo
-Identificar as tentativas de log invalidas e filtrar os seus IP's.

Como faz?

-Lê o arquivo auth.log

-Detecta tentativas de login falhadas

-Extrai endereços IP

-Conta quantas vezes cada IP aparece

-Identifica IPs suspeitos (várias tentativas)

 Linguagem usada

 -pyhton

 Ferramentas usadas
 
 -linha de comandos do windows
 
 -python

 OBS:Este projecto foi criad com base nos conhecimentos adquiridos no curso de Cibersegurança do Google.
